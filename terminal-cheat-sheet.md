# Cheats for Terminal

- # Ordner erstellen
    > mkdir (make directory)

        Beispiel: mkdir Ordner-1


- # Datei erstellen
    > touch

        Beispiel: Datei-1.txt/html/md/etc...


- # Anzeige des Aktuellen verzeichnisses
    > pwd (print working directory)


- # Löschen von Dateien
    >rm

        Beispiel: rm Datei.txt/html/md/etc...


- # Löschen von Ordnern
    > rm -d 

        Beispiel: rm -d Ordner-1


- # Löschen von Ordner mit Dateien
    > rm -r

       Beispiel: rm -r Ordner-1


- # Mehrere Datein vom selben typen löschen
    > rm *

        Beispiel: rm *.js/html/md/txt/etc...


- # Alle Dateien in einem Ordner Löschen
    >rm *


- # Auflistung von Ordner und Dateien
    > ls (list)

- # Auflistung von Ordner und Dateien mit mehr  Info
    >ls -l


- # Auflistung von allen(Versteckten) Dateien
    >ls -a


- # Auflistung von allen(versteckten) Dateien und mehr info
    > ll


- # Datei in Visual Studios öffnen
    > code 

        Beispiel: code Datei-1.md/html/css/txt/etc... 


- # Visual Studios öffnen
    >code


- # Arbeitsverzeichnis Wechseln
    > cd (change directory) 

        Beispiel: cd Ordner-1



- # Datei Kopieren und umbenennen
    > cp (copy)

        Beispiel: cp Datei.txt Datei-abc.txt
                  

- # Ordner Kopieren und Verschieben
    > cp -r 
        
        Beispiel:cp -r Datei.txt Ordner-1


- ## Datei in einen anderen Ordner Kopieren
    > cp ../

        Beispiel: cp Datei.txt ../Ordner-1


- # Datei Verschieben
    > mv

        Beispiel: mv Datei.txt Ordner

    
- ## Datei Verschieben und Umbenennen
    >mv

        Beispiel: mv Datei.txt Ordner/Datei-Neu.txt


- ## Datei Umbenennen
    > mv

        Beispiel: mv Datei.txt Datei-Neu.txt


- ## Datei in den Vorherigen Ordner Verschieben
    >mv

        Beispiel: mv Datei-1.txt ../

- # Öffnet im Terminal eine Datei
    >less
        
         Beispiel: less Datei.txt/html/md/etc...

- # Adminrechte, Installieren, Updaten...
    >sudo (Superuser)

- # Handbuch öffnen
    >man 

        Beispiel: man ls oder man history etc...


- # Zeigt den Verlauf
    >history

    
- ## Löscht den Verlauf
    >history -c

- # Öffnet den home Ordner
    >nautilus


- # Datei in Editor erstellen und öffnen
    >nano

        Nano short-cuts:
        strg+k = Ausschneiden
        strag+u = Kopieren
        strg+g = Hilfe in nano
        strg+b = Bewegt courser einen Buchstaben zurück
        strg+f = Bewegt den courser einen Buchstaben vorwärts
        strg+c = Position vom courser
        strg+f = Neue zeile
        strg+o = Speichern und neuen Namen eingeben
        strg+s = Speichern
